## 快速开始

登录后会进入工作台。工作台会展示可用的助手卡片，你可以直接选择一个助手开始对话，也可以点击“创建专属 Agent”准备一个更贴合某类任务的专属助手。

常见使用方式：

- 想随手问问题、整理内容、写材料：进入“新对话”或选择默认助手。
- 想让不同助手处理不同类型的任务：在工作台选择对应助手，或创建专属 Agent。
- 想让助手参考你的资料：先到“知识库”上传文档，再回到对话中提问。
- 想定期执行提醒、日报、检查类任务：到“定时任务”创建自动执行计划。

## 对话

点击左侧“新对话”可以开始一次新的沟通。输入框中写下你的目标、问题或要处理的内容，然后点击发送。

如果你还没有选择具体 Agent，系统会使用默认助手。若你希望某次对话由特定 Agent 处理，可以在发送前选择 Agent。对话开始后，这个会话会固定使用当前 Agent，方便后续保持同一上下文。

发送附件时，可以把文件或图片加入当前消息。建议在文字里说明你希望助手如何使用这些附件，例如“总结这份文档的重点”“根据图片内容写一段说明”。

## 会话列表

左侧会按 Agent 和普通对话整理历史会话。点击会话名称即可继续之前的沟通。

会话旁出现未读提示时，表示助手有新的回复。你打开对应会话后，未读状态会自动清除。

如果某个会话名称不方便识别，可以在会话菜单中重命名。删除会话前请确认不再需要其中内容，删除后无法恢复。

## Agent

Agent 是可复用的专属助手。你可以为它设置名称、头像和说明，让它更稳定地服务某类场景，例如写作、项目管理、研究整理或客服回复。

创建 Agent 的建议：

- 名称保持清楚，方便在工作台和会话列表中识别。
- 说明写成使用者能理解的目标和风格，不需要写复杂规则。
- 一个 Agent 只负责一类主要任务，效果通常更稳定。

系统内置 Agent 可以查看，但不建议修改。用户自定义 Agent 可以在“设置 > Agents”里管理。

## 知识库

知识库用于保存希望助手参考的资料。进入“知识库”后，先选择对应 Agent，再上传文件或创建文件夹整理资料。

使用建议：

- 优先上传 Markdown 文档，便于助手检索和引用。
- 文件名尽量能表达内容，例如“产品介绍.md”“客户问答.md”。
- 按主题建立文件夹，避免所有资料混在一起。
- 更新资料后，可以在对话中明确告诉助手参考哪些内容。

知识库适合放长期资料，例如产品说明、业务流程、常见问题、写作规范和项目背景。

## 记忆

“记忆”页面用于查看当前账号已经保存的重要信息，也可以写入一条测试记忆，检查后续是否能被找到。

你可以把偏好、长期背景、常用事实放进记忆，例如“我偏好简洁的中文回答”“我的项目主要面向企业内部用户”。这些内容会帮助助手在后续互动中更贴近你的习惯。

如果某条记忆不再准确，建议及时更新相关内容，避免助手继续参考过期信息。

## 技能商店

技能用于扩展助手能处理的任务类型。进入“技能商店”后，可以查看当前可用技能，也可以按需要安装、上传或导入新技能。

使用技能时，建议先确认它适合当前 Agent 的工作范围。不要给一个 Agent 安装太多无关技能，以免后续选择和维护变得混乱。

## 定时任务

定时任务可以让 Agent 在指定时间自动收到一段消息并执行，适合日报、提醒、周期性检查和固定流程。

创建任务时需要填写：

- 任务名称：方便以后识别，例如“每日上午简报”。
- 任务消息：Agent 到时间后要处理的具体内容。
- 调度方式：可以选择固定间隔、Cron 或单次执行。

创建后可以在列表中启用、停用、立即执行或删除任务。建议先用“立即执行”确认任务消息符合预期，再长期启用。

## 外观设置

在“设置 > 外观”中可以切换主题、强调色、侧边栏效果、界面密度和文字对比度。调整会立即生效，你可以根据屏幕尺寸和阅读习惯选择更舒适的显示方式。

如果调整后不满意，可以点击“恢复默认外观”回到初始效果。

## 使用建议

为了让助手更准确地完成任务，尽量在消息中说明目标、背景、输入资料和期望输出。比如：

- “请把这段会议记录整理成待办清单，按负责人分组。”
- “参考知识库里的产品说明，写一版给客户看的介绍，语气自然一点。”
- “每天上午 9 点提醒我查看昨日销售数据，并列出异常项。”

一次只交代一个清晰目标，通常比把多个互不相关的要求放在同一条消息里更容易得到好结果。
